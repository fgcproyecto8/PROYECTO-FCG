# Programación - Sistema de Gestión de Partidos de Fútbol 5

## Descripción del Proyecto

Este proyecto consiste en una plataforma digital orientada a facilitar la organización de partidos de fútbol 5. El sistema permite conectar jugadores con canchas disponibles, optimizando la coordinación de encuentros deportivos mediante una interfaz centralizada.

---

## Objetivo Principal

Se proyecta desarrollar una aplicación que permita a los usuarios organizar partidos, unirse a encuentros existentes y gestionar reservas de canchas. Asimismo, se busca brindar a los dueños de canchas una herramienta para administrar sus espacios, horarios y disponibilidad, centralizando toda la información en un único sistema.

---

## Funcionalidades del Sistema

### 👤 Módulo del Usuario (Jugador)

- **Gestión de Perfil:** Espacio personal con información del jugador (nombre, posición, partidos jugados, etc.).  
- **Visualización de Canchas:** Acceso a las canchas disponibles con sus horarios.  
- **Creación de Partidos:** Posibilidad de organizar partidos de fútbol 5 (5 vs 5).  
- **Unión a Partidos:** Permite unirse a partidos existentes hasta completar los cupos.  
- **Historial de Participación:** Consulta de los partidos en los que participó.  

---

### 🏟️ Módulo del Dueño de Cancha

- **Publicación de Canchas:** Registro y gestión de canchas dentro del sistema.  
- **Gestión de Horarios:** Definición de disponibilidad de turnos.  
- **Administración de Precios:** Configuración de tarifas por horario o cancha.  
- **Visualización de Reservas:** Acceso a los partidos programados en sus instalaciones.  

---

### 🛠️ Módulo Administrador

- **Gestión de Usuarios:** Control y administración de cuentas de jugadores y dueños.  
- **Supervisión del Sistema:** Monitoreo de actividad general.  
- **Gestión de Contenido:** Validación o eliminación de canchas o datos incorrectos.  
- **Control de Integridad:** Asegurar el correcto funcionamiento del sistema.  

---

## Tecnologías Utilizadas

- Backend: Django (Python)  
- Frontend: React + Vite  
- Base de Datos: SQLite  
- Control de Versiones: Git y GitHub
